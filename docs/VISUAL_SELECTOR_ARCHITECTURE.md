## Component Responsibilities

### Frontend (Vue.js)

**NodeConfigPanel.vue**
- Renders "Visual Selector" button
- Manages session state (loading, active, error)
- Polls for selected fields
- Imports fields into node configuration
- Provides visual feedback to user

**selector.ts (API Client)**
- HTTP requests to backend
- Polling helper function
- Type definitions
- Error handling

### Backend (Go)

**SelectorHandler**
- HTTP endpoint handlers
- Request/response validation
- Error responses
- Session lifecycle coordination

**ElementSelectorManager**
- Session creation and management
- Browser acquisition and release
- Field polling from browser
- Automatic cleanup
- Concurrent session support

**BrowserPool**
- Headed browser creation
- Browser lifecycle management
- Separation of pooled vs headed browsers
- Resource cleanup

### Browser (Chromium + Vue.js)

**Injected Overlay Application**
- Vue 3 reactive UI
- Element highlighting
- Selector generation
- User interactions
- Data storage in window object
